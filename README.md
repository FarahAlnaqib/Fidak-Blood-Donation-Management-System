# 🩸 Fidak - Blood Donation Management System

A web-based system for managing blood donation processes, including donor records, blood inventory, and request handling.

## 🚀 Features
- Donor management  
- Blood inventory tracking  
- Request system  
- Admin dashboard  

## 🛠️ Tech Stack
PHP • MySQL • HTML • CSS • JavaScript

Developed as part of my Computer Science studies to apply web development and database concepts in a real-world system.
